# ProbeShell

## 是什么?

Grafana + VictoriaMetrics 被控端一键安装脚本

## 快速开始
请使用`root`用户运行如下的命令从而使用脚本,卸载同样适用

```bash
bash <(curl -sSL https://raw.githubusercontent.com/Sm1rkBoy/ProbeShell/main/agent.sh)
```
